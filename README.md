# 🥑 Vegan & Cheese

**Vegan & Cheese** is a lightweight recipe website and publishing platform built with **Jekyll** and **GitHub Pages**.

The goal is simple:

> **Create beautiful recipe websites that are fast, inexpensive to host, easy to maintain, and easy to publish to.**

---

## Features

* 📖 Beautiful recipe pages
* 🔍 Site-wide recipe search
* 🖼️ Image galleries
* 🏷️ Categories and tags
* 📱 Mobile-friendly design
* ⚡ Static site performance via GitHub Pages
* 📝 Browser-based Recipe CMS
* 🚀 One-click publishing using a GitHub Publish Key

---

## Technology

* Jekyll
* GitHub Pages
* HTML / CSS / JavaScript
* Markdown recipes
* GitHub REST API

## Adding a Recipe

Visit:

```
https://your-blog-name.com/new-recipe/
```

Fill out:

* Title
* Main picture
* Ingredients
* Directions
* Gallery
* Categories

Click **Publish**, enter your **Food Blog Publish Key**, and the recipe is committed directly to the repository.

GitHub Pages automatically rebuilds the site and publishes the new recipe.

---

## Philosophy

This project is intentionally built around open standards:

* Recipes are stored as Markdown.
* Images remain in the repository.
* Content is portable.
* No proprietary database.
* No lock-in.

The long-term vision is to evolve this project into an easy-to-use blogging and recipe publishing platform where creators own both their content and their domain name.
